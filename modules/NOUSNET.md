# Module: NousNet

Identity-first encrypted overlay network.

## Initial transport

Start with QUIC or TCP local prototype. Add Noise-style encryption later. Add Tor adapter later.

## Address model

```text
nous://<identity>/<service>/<object>
```

## Modes

```text
direct LAN
internet relay
Tor/onion transport
store-and-forward offline sync
Bluetooth/LoRa/local mesh later
```

## Rule

Tor is optional privacy transport, not default transport.
